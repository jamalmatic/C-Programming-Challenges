# Challenge 03 — Average of Three Numbers

## Description

Write a C program that reads **three numbers** from the user and calculates their **average**.

The program should display the result formatted with **two decimal places**.

---

## Formula


Average = (a + b + c) / 3


---

## Example

Input


10 20 30


Output


Average = 20.00


---

## Input

Three numbers entered by the user.

Input can be given:

**Space separated**


15 25 35


or **line separated**


15
25
35


---

## Output

Display the calculated average.

Example:


Average = 25.00


The result should be printed with **two decimal places**.

---

## Concepts Practiced

This challenge helps you practice:

- reading multiple inputs
- using `integer` and `float` variables
- arithmetic operations
- calculating averages
- formatted output using `printf`

---
