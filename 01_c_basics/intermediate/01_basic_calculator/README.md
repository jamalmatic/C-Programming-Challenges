# Challenge 01 — Basic Calculator

## Description

Write a C program that performs basic arithmetic operations on two numbers.

The program should calculate:

- Addition
- Subtraction
- Multiplication
- Division

The result of each operation should be displayed clearly.

---

## Input

Two numbers provided by the user.

Input can be given either:

- on the **same line separated by space**
- on **separate lines**

Example:

10 5

or

10
5

---

## Output

Display the result of the four arithmetic operations.

Example output:

Addition = 15.00
Subtraction = 5.00
Multiplication = 50.00
Division = 2.00

Result should be displayed with **two decimal places**.

---

## Concepts Practiced

This challenge helps you practice:

- Variables
- Arithmetic operators (`+ - * /`)
- Input using `scanf`
- Output formatting using `printf`
- Working with floating point numbers

---

## Hints

- Use `printf("%.2f")` to format the division result.
- Read two numbers using `scanf`.
