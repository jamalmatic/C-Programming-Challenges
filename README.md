# C Programming Challenges

A curated collection of **C programming challenges** organized from **beginner to advanced**.

This repository is designed to help learners progressively improve their C programming skills through **structured topics and practical exercises**.

It can be used by:

* Beginners learning C step by step
* Developers who want to practice algorithms in C
* Students preparing for technical interviews
* Anyone who wants to strengthen low-level programming skills

---

# Project Structure

The repository is organized by **topic**.
Each topic contains explanations and challenges divided into three difficulty levels:

* Beginner
* Intermediate
* Advanced

```
c-challenges/
│
├── README.md
│
├── 01_c_basics/
├── 02_control_flow/
├── 03_functions/
├── 04_recursion/
├── 05_arrays/
├── 06_strings/
├── 07_pointers/
├── 08_structs/
├── 09_dynamic_memory/
├── 10_data_structures/
├── 11_algorithms/
└── 12_system_programming/
```

Each topic folder contains:

```
topic/
│
├── README.md          → Explanation of the concept
│
├── beginner/
├── intermediate/
└── advanced/
```

Each challenge has its own folder:

```
challenge_name/
│
├── README.md          → Problem description
├── solution.c         → Reference implementation
└── test_cases.txt
```

# Learning Roadmap

## 01. C Basics

Introduction to the C language.

Topics:

* Program structure
* Variables and data types
* Input / Output
* Operators
* Type conversion

---

## 02. Control Flow

Topics:

* `if` / `else`
* `switch`
* `for`, `while`, `do-while`
* `break` and `continue`

---

## 03. Functions

Topics:

* Function declaration
* Parameters
* Return values
* Scope and modular programming

---

## 04. Recursion

Topics:

* Recursive functions
* Base cases
* Stack behavior

---

## 05. Arrays

Topics:

* One dimensional arrays
* Two dimensional arrays
* Basic array algorithms

---

## 06. Strings

Topics:

* Character arrays
* String manipulation
* `<string.h>` library

---

## 07. Pointers

Topics:

* Pointer basics
* Pointer arithmetic
* Passing variables by reference

---

## 08. Structures

Topics:

* `struct`
* `enum`
* Arrays of structures

---

## 09. Dynamic Memory

Topics:

* `malloc`
* `calloc`
* `realloc`
* `free`

---

## 10. Data Structures

Topics:

* Linked lists
* Stack
* Queue
* Trees

---

## 11. Algorithms

Topics:

* Sorting algorithms
* Searching algorithms
* Complexity basics

---

## 12. Systems Programming

Advanced C topics:

* File I/O
* Bit manipulation
* Command line tools
* Low level memory operations
